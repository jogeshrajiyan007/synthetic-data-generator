# 🚀 AXA GovernGPT – Next.js Application Prompt

You are a senior full-stack developer. Build a modern, production-ready Next.js application named **AXA GovernGPT**, a Data Governance RAG (Retrieval-Augmented Generation) platform for AXA Insurance.

---

## 🧠 Application Overview

**AXA GovernGPT** is an AI-powered data governance assistant that allows users to:
- Upload enterprise data sources (documents, structured files, images)
- Interact with an AI agent to query, understand, and govern data

---

## 🛠️ Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- ShadCN UI
- Zustand (or React Context)
- React Dropzone (for uploads)

---

## 🧩 Layout

### Two-panel responsive layout:

| Left Panel (Sources) | Right Panel (Chat) |
|---------------------|--------------------|
| 25–30% width        | 70–75% width       |

---

## 📂 LEFT PANEL — Sources

### Header
**AXA GovernGPT • Sources**

---

### 🔢 Source Limit

- Maximum allowed sources: **20 files**
- Prevent uploads beyond limit
- Show error message if exceeded

---

### 📊 Source Progress Indicator

Add a progress bar at the top:

- Display: `X / 20 Sources`
- Dynamic fill based on number of uploaded files

#### Color Rules:
- 0–70% → primary color
- 70–90% → warning (yellow/orange)
- 90–100% → danger (red)

---

### 📥 Upload Features

#### Supported File Types:
- Documents: `pdf, docx, doc, txt, md`
- Data: `csv, xlsx, xls, json, xml`
- Images: `jpg, jpeg, png`

#### Features:
- Drag & drop upload
- Click to upload
- File list display:
  - Name
  - Type icon
  - Size
  - Delete button

#### Optional:
- Upload progress per file
- Image preview
